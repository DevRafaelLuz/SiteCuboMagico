MOVIMENTOS BÁSICOS

Para solucionar o Cubo Mágico nós vamos utilizar algumas fórmulas, que são sequencias de
movimentos que podem ser escritos (traduzidos) com letras. Parece muito complexo,
mas é extremamente simples e intuitivo.

Cada lado do cubo é representado pela sua inicial em inglês. Se a letra estiver sozinha
significa que o movimento é no sentido horário. Se estiver acompanhada de um apóstrofo,
significa que é no sentido anti-horário. E se for seguida do número 2, significa que é
um movimento duplo.

R

Right / direita horário.

R'

Right / direita anti-horário.

F

Front / face horário.

F'

Front / face anti-horário.

D

Down / base horário.

D'

Down / base anti-horário.

L

Left / esquerda horário.

L'

Left / esquerda anti-horário.

B

Back / atrás horário.

B'

Back / atrás anti-horário.

U

Up / topo horário.

U'

Up / topo anti-horário.


OBS: Se houver um número ao lado da letra, significa a quantidade de vezes que a camada deve ser rotacionada.
